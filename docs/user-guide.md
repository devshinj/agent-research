# 1. 시작하기

## 1.1 회원가입

1. 접속 주소로 이동하면 **로그인 화면**이 나타납니다.
2. 하단의 **"가입하기"** 링크를 클릭합니다.
3. 아래 항목을 입력합니다:
   - **이메일** — 로그인에 사용할 이메일 주소
   - **닉네임** — 랭킹 등에 표시될 이름
   - **비밀번호** — 8자 이상
   - **초대 코드** — 관리자가 초대 코드 제도를 활성화한 경우에만 표시됩니다
4. **"가입하기"** 버튼을 누르면 계정이 생성되고, 초기 자금이 자동으로 지급됩니다.

> 📸 **[스크린샷: 회원가입 화면]**
>
> _여기에 회원가입 폼 스크린샷을 추가하세요_

---

## 1.2 로그인

1. **이메일**과 **비밀번호**를 입력합니다.
2. **"로그인"** 버튼을 클릭합니다.
3. 로그인 성공 시 대시보드로 이동합니다.

> 📸 **[스크린샷: 로그인 화면]**
>
> _여기에 로그인 폼 스크린샷을 추가하세요_

💡 **참고:** 세션은 일정 시간이 지나면 자동으로 갱신됩니다. 장시간 미사용 시 다시 로그인이 필요할 수 있습니다.

# 2. 대시보드

로그인 후 가장 먼저 보이는 화면입니다. 내 포트폴리오 현황을 한눈에 파악할 수 있습니다.

## 2.1 주요 지표 (KPI 카드)

화면 상단에 4개의 핵심 지표가 카드 형태로 표시됩니다:

| 지표 | 설명 |
|------|------|
| **총 자산** | 보유 현금 + 보유 코인 평가금액의 합계 |
| **가용 현금** | 현재 매수에 사용할 수 있는 잔고 |
| **총 손익** | 전체 투자 손익 금액 (원화) |
| **수익률** | 초기 자금 대비 수익률 (%) |

오른쪽 상단에는 **트레이딩 상태 뱃지**가 표시됩니다:
- 🟢 **활성** — 자동매매가 작동 중
- 🔴 **비활성** — 자동매매가 꺼져 있음

> 📸 **[스크린샷: 대시보드 KPI 카드 영역]**
>
> _여기에 KPI 카드 4개와 상태 뱃지가 보이는 스크린샷을 추가하세요_

---

## 2.2 자산 추이 차트

KPI 카드 아래에 **총 자산의 변화 추이**가 영역 차트로 표시됩니다.

기간 선택 버튼:
- **24시간** | **1일** | **1주** | **1개월**

원하는 기간을 클릭하면 해당 범위의 자산 변동 그래프가 표시됩니다.

> 📸 **[스크린샷: 자산 추이 차트]**
>
> _여기에 기간 탭과 차트가 보이는 스크린샷을 추가하세요_

---

## 2.3 보유 포지션

현재 보유 중인 코인 목록이 테이블로 표시됩니다.

| 컬럼 | 설명 |
|------|------|
| **종목** | 코인 마켓 코드 (예: KRW-BTC) |
| **수량** | 보유 수량 |
| **평균 매수가** | 평단가 |
| **현재가** | 실시간 시세 |
| **평가 손익** | 현재 미실현 손익 |
| **모드** | AUTO(자동) / MANUAL(수동) — 클릭하여 전환 가능 |

### 포지션 상세 보기
테이블의 행을 클릭하면 해당 코인의 **캔들스틱 차트**가 펼쳐집니다.

### 트레이드 모드 전환
- **AUTO** — AI 시그널에 따라 자동으로 매도 실행
- **MANUAL** — 사용자가 직접 매도 타이밍을 결정

모드 뱃지를 클릭하면 전환할 수 있습니다.

> 📸 **[스크린샷: 보유 포지션 테이블 + 확장된 캔들 차트]**
>
> _여기에 포지션 테이블과 한 행이 확장된 모습의 스크린샷을 추가하세요_

---

## 2.4 거래 내역

대시보드 하단에 **최근 거래 내역**이 페이지네이션과 함께 표시됩니다.

| 컬럼 | 설명 |
|------|------|
| **시간** | 거래 체결 시각 |
| **종목** | 마켓 코드 |
| **구분** | BUY(매수) / SELL(매도) |
| **사유** | ML_SIGNAL, MANUAL, STOP_LOSS, TAKE_PROFIT 등 |
| **가격** | 체결 가격 |
| **수량** | 거래 수량 |

> 📸 **[스크린샷: 거래 내역 테이블]**

# 3. 거래소 (수동 매매)

좌측 사이드바에서 **"거래소"**를 클릭하면 수동 매매 화면으로 이동합니다.

## 3.1 마켓 목록

왼쪽 패널에 업비트 마켓 목록이 실시간으로 표시됩니다.

**필터 탭:**
- **전체** — 모든 마켓
- **스크리닝** — AI가 관심 종목으로 선정한 마켓 (뱃지 표시)
- **보유** — 현재 보유 중인 종목만

**정렬 옵션:** 종목명 | 현재가 | 전일 대비 | 거래량

각 행에는 실시간 가격이 표시되며, 가격 변동 시 **깜빡임 애니메이션**이 적용됩니다.

> 📸 **[스크린샷: 거래소 마켓 목록 패널]**
>
> _여기에 마켓 목록과 필터 탭이 보이는 스크린샷을 추가하세요_

---

## 3.2 마켓 상세 및 차트

마켓 목록에서 종목을 선택하면 오른쪽에 **상세 정보**가 표시됩니다:
- **캔들스틱 + 거래량 차트**
- 타임프레임 선택: **1분** | **5분** | **15분** | **1시간** | **4시간** | **1일**

> 📸 **[스크린샷: 마켓 상세 캔들스틱 차트]**
>
> _여기에 종목 선택 후 표시되는 캔들 차트 스크린샷을 추가하세요_

---

## 3.3 매수 주문

1. 마켓 목록에서 매수하고 싶은 종목을 선택합니다.
2. **매수 금액(KRW)**을 입력합니다.
   - 최소 주문 금액이 있으므로 너무 소액은 불가합니다.
3. 필요 시 **손절가(Stop-Loss)**와 **익절가(Take-Profit)**를 설정합니다.
4. **"매수"** 버튼을 클릭합니다.
5. **확인 모달**이 나타나면 내용을 확인하고 **"확인"**을 누릅니다.

> 📸 **[스크린샷: 매수 주문 폼 + 확인 모달]**
>
> _여기에 매수 폼과 확인 팝업 스크린샷을 추가하세요_

---

## 3.4 매도 주문

1. 보유 중인 종목을 마켓 목록에서 선택합니다.
2. **매도 비율**을 설정합니다 (0~1 사이, 예: 0.5 = 50% 매도).
3. **"매도"** 버튼을 클릭합니다.
4. 확인 모달에서 **"확인"**을 누릅니다.

💡 **팁:** 매도 비율 `1`을 입력하면 전량 매도됩니다.

> 📸 **[스크린샷: 매도 주문 폼]**
>
> _여기에 매도 폼 스크린샷을 추가하세요_

---

## 3.5 실시간 알림

거래가 체결되면 화면에 **실시간 알림**이 나타납니다:
- 종목명, 매수/매도 구분, 체결 가격, 사유가 표시됩니다.
- 이 알림은 WebSocket을 통해 즉시 전달됩니다.

# 4. 전략 (AI 시그널)

좌측 사이드바에서 **"전략"**을 클릭합니다. AI 모델이 분석한 결과를 확인할 수 있습니다.

## 4.1 스크리닝 결과

AI가 거래량, 변동성 등의 조건을 충족하는 종목을 자동으로 선정합니다.

| 컬럼 | 설명 |
|------|------|
| **종목** | 마켓 코드 |
| **현재가** | 실시간 가격 |
| **거래량** | 24시간 거래량 |
| **변동성** | 최근 변동성 (%) |
| **스코어** | AI 종합 점수 (높을수록 관심도 높음) |

> 📸 **[스크린샷: 스크리닝 테이블]**
>
> _여기에 스크리닝 결과 테이블 스크린샷을 추가하세요_

---

## 4.2 트레이딩 시그널

AI 모델이 생성한 매매 시그널이 표시됩니다:

| 시그널 | 의미 |
|--------|------|
| 🟢 **BUY** | 매수 추천 |
| 🔴 **SELL** | 매도 추천 |
| ⚪ **HOLD** | 관망 |

각 시그널에는 **신뢰도 바**가 함께 표시됩니다. 신뢰도가 높을수록 모델이 해당 판단에 확신을 가지고 있다는 의미입니다.

### 시그널 근거 확인
시그널에 마우스를 올리면 **SHAP 값 기반의 설명 툴팁**이 나타납니다. 어떤 기술적 요인이 시그널에 기여했는지 확인할 수 있습니다.

> 📸 **[스크린샷: 시그널 목록 + SHAP 툴팁]**
>
> _여기에 시그널 카드와 마우스 호버 시 나타나는 SHAP 설명 스크린샷을 추가하세요_

---

## 4.3 모델 상태

각 종목별 ML 모델의 현재 상태를 확인할 수 있습니다:
- **정확도** — 모델의 예측 정확도
- **최근 학습일** — 마지막으로 모델이 학습된 시점
- **시그널 분포** — BUY/HOLD/SELL 비율 막대 차트
- **학습 중 표시** — 현재 재학습이 진행 중인 경우 펄싱 뱃지 표시

> 📸 **[스크린샷: 모델 상태 카드]**
>
> _여기에 모델 상태 카드 스크린샷을 추가하세요_

# 5. 리스크 관리

좌측 사이드바에서 **"리스크"**를 클릭합니다. 투자 위험을 관리하는 설정 화면입니다.

## 5.1 리스크 현황

화면 상단에 현재 리스크 상태가 표시됩니다:

| 지표 | 설명 |
|------|------|
| **서킷 브레이커** | 🟢 정상 / 🔴 발동 (발동 시 거래 일시 중지) |
| **일일 손실률** | 오늘 누적 손실 비율 (진행 바) |
| **연속 손실** | 연속 손절 횟수 (리스크 미터) |
| **일일 거래 수** | 오늘 체결된 총 거래 수 (진행 바) |

서킷 브레이커가 발동되면 설정된 **쿨다운 시간** 동안 자동매매가 중단됩니다.

> 📸 **[스크린샷: 리스크 현황 패널]**
>
> _여기에 서킷 브레이커 상태와 리스크 미터 스크린샷을 추가하세요_

---

## 5.2 개인 리스크 설정

모든 사용자가 조정할 수 있는 **개인 설정** 항목입니다:

| 설정 | 설명 | 기본값 참고 |
|------|------|-------------|
| **손절 비율 (%)** | 이 비율만큼 하락하면 자동 매도 | 예: 3% |
| **익절 비율 (%)** | 이 비율만큼 상승하면 자동 매도 | 예: 5% |
| **트레일링 스톱 (%)** | 고점 대비 이 비율만큼 하락하면 매도 | 예: 1.5% |
| **일일 최대 손실 (%)** | 하루 최대 허용 손실 (초과 시 서킷 브레이커) | 예: 5% |
| **포지션 비율 (%)** | 1회 매수 시 총 자산 대비 투자 비율 | 예: 10% |
| **최대 포지션 수** | 동시에 보유 가능한 종목 수 | 예: 5개 |

각 항목은 **슬라이더**로 조정하며, 최소/최대 범위가 제한되어 있습니다.

### 설정 변경 방법
1. 원하는 항목의 슬라이더를 조정합니다.
2. **"적용"** 버튼을 클릭합니다.
3. 성공 시 알림이 표시됩니다.

💡 **"초기화"** 버튼을 누르면 기본값으로 되돌릴 수 있습니다.

> 📸 **[스크린샷: 개인 리스크 설정 슬라이더]**
>
> _여기에 슬라이더 설정 패널 스크린샷을 추가하세요_

# 6. 랭킹

좌측 사이드바에서 **"랭킹"**을 클릭합니다. 전체 사용자의 투자 성과 순위를 확인할 수 있습니다.

## 6.1 리더보드

수익률 기준으로 정렬된 전체 사용자 랭킹이 표시됩니다.

| 컬럼 | 설명 |
|------|------|
| **순위** | 🥇 🥈 🥉 + 번호 |
| **닉네임** | 사용자 닉네임 (본인은 **"나"** 뱃지 표시) |
| **총 자산** | 현재 총 자산 (KRW) |
| **손익** | 누적 실현 손익 |
| **수익률** | 초기 자금 대비 수익률 |
| **승률** | 수익으로 종료된 거래 비율 |
| **거래 수** | 총 체결 거래 횟수 |
| **최대 낙폭** | 최고점 대비 최대 하락 폭 |
| **추이** | 30일간 자산 변화 스파크라인 차트 |

각 컬럼 헤더에 마우스를 올리면 **설명 툴팁**이 표시됩니다.

> 📸 **[스크린샷: 랭킹 리더보드]**
>
> _여기에 랭킹 테이블과 스파크라인 스크린샷을 추가하세요_

# 7. 시스템

좌측 사이드바에서 **"시스템"**을 클릭합니다.

## 7.1 자동매매 제어

| 항목 | 설명 |
|------|------|
| **자동매매 ON/OFF** | AI 자동매매를 켜거나 끕니다 |

자동매매를 **OFF**로 설정해도 기존 보유 포지션의 손절/익절은 계속 작동합니다. 새로운 매수 진입만 중단됩니다.

> 📸 **[스크린샷: 시스템 페이지 자동매매 토글]**
>

# 8. 자주 묻는 질문

### Q. 실제 돈이 사용되나요?
**아닙니다.** 이 시스템은 모의투자(Paper Trading)입니다. 실제 업비트 시세를 사용하지만, 모든 거래는 가상으로 처리됩니다.

---

### Q. 자동매매는 어떻게 동작하나요?
AI 모델이 업비트의 실시간 시세 데이터를 분석하여 **BUY/SELL 시그널**을 생성합니다. 자동매매가 활성화되어 있고 리스크 관리 조건을 통과하면 시그널에 따라 자동으로 주문이 실행됩니다.

---

### Q. 자동매매와 수동매매를 동시에 사용할 수 있나요?
**네.** 거래소 페이지에서 언제든 수동으로 매수/매도할 수 있습니다. 또한 보유 포지션별로 AUTO/MANUAL 모드를 개별 설정할 수 있어, 특정 포지션만 수동 관리하는 것도 가능합니다.

---

### Q. 서킷 브레이커가 발동되면 어떻게 되나요?
일일 최대 손실 초과, 연속 손절 횟수 초과, 일일 최대 거래 수 초과 중 하나라도 해당되면 **서킷 브레이커가 발동**됩니다. 설정된 쿨다운 시간(기본 몇십 분) 동안 새로운 매수 진입이 자동으로 중단됩니다. 쿨다운이 끝나면 자동으로 해제됩니다.

---

### Q. 수수료와 슬리피지가 적용되나요?
**네.** 실제 업비트와 유사하게 매매 수수료(0.05%)와 슬리피지(0.05%)가 적용됩니다. 모의투자이지만 현실적인 환경을 제공하기 위함입니다.

---

### Q. 초기 자금은 얼마인가요?
관리자가 설정한 금액이 자동으로 지급됩니다. 정확한 금액은 가입 후 대시보드에서 확인할 수 있습니다.

---

### Q. 화면이 실시간으로 업데이트되지 않아요.
WebSocket 연결이 끊어졌을 수 있습니다. **페이지를 새로고침**하세요. 문제가 지속되면 다시 로그인해 보세요.

---

### Q. 랭킹은 어떤 기준으로 정렬되나요?
**수익률(%)** 기준으로 정렬됩니다. 같은 수익률인 경우 총 손익 금액이 높은 사용자가 상위에 표시됩니다.

---

### Q. 포지션의 추가매수(물타기)는 어떻게 동작하나요?
자동매매 모드에서 보유 중인 종목의 가격이 설정된 비율만큼 하락하면 **자동으로 추가 매수**가 실행됩니다. 추가매수 횟수는 제한되어 있으며, 대시보드 포지션 테이블에서 추가매수 횟수를 확인할 수 있습니다.

---
